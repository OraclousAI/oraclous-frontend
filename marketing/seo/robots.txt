# robots.txt — oraclous.com (marketing site)
# ---------------------------------------------------------------------------
# This file governs the PUBLIC, INDEXABLE marketing surface at oraclous.com.
# The product console at app.oraclous.com is a SEPARATE origin with its OWN
# robots.txt — it is auth-walled and excluded from indexing entirely
# (see the "Cross-domain note" at the bottom of this file).
#
# Policy: maximise AI-search visibility. Oraclous's strategy is to be the
# brand that AI answer engines cite for "governed / sovereign / open-source
# agentic AI". AI crawlers are therefore EXPLICITLY ALLOWED here rather than
# left to inherit the wildcard rule, because (a) explicit Allow blocks remove
# any ambiguity and (b) it documents the intent for future maintainers.
#
# Rubric ties: geo-technical §1.1 (robots.txt validity), §1.2 (AI crawler
# access — CRITICAL), §1.3 (sitemap reference); geo-crawlers Tier 1/2/3.
# ---------------------------------------------------------------------------
